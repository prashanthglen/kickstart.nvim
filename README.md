# kickstart.nvim

This is my personal fork of [kickstart.nvim](https://github.com/nvim-lua/kickstart.nvim). Apart from the base setup I have added here:

- [ToggleTerm](https://github.com/akinsho/toggleterm.nvim) for terminal support.
- [blink.nvim](https://github.com/saghen/blink.nvim) for autocomplete.
- [Markdown Preview](https://github.com/iamcco/markdown-preview.nvim).
    - Requires `yarn` to be installed
    - Requires you to run `yarn global add tslib`
    - Required setting environment variable `NODE_OPTIONS="--no-experimental-webstorage" for Nodejs 25.2.0
- Enabled Neo-Tree plugin.
